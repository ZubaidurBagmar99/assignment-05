What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model. DOM is a programming API for HTML and XML documents. We can also add events to these elements to make our page more dynamic.It is a programming interface that allows us to create, change or remove elements from the document.
The purpose of DOM. It explain:

  1. The HTML elements as objects
  2. The methods to access all HTML elements
  3. The properties of all HTML elements
  4. The events for all HTML elements
  5. That is: The HTML DOM is a standard for how to get, change, add, or delete HTML elements.

How will you select HTML elements using DOM? Name the DOM methods.

There are 5 ways select elements in a DOM using selectors.

  1. getElementsByTagName()
  2. getElementsByClassName()
  3. getElementById()
  4. querySelector()
  5. querySelectorAll().

What is event bubble?

Event bubbling is a concept in the DOM. In the event bubbling model, an event starts at the most specific element and then flows upward toward the least specific element.

What is a callback function and why will you use it?

A callback function in javascript is a function that is passed as an argument in another function.
A callback's purpose is to execute code in response to an event.
